# 蒙哥马利县雇员工资分析项目

## 项目简介
本项目对蒙哥马利县2020年雇员工资数据进行全面分析，包括工资统计、部门分析、工资组成和级别关系研究。

## 文件说明

### 数据文件
- `Montgomery_County_Employee_Salaries_2020.csv` - 原始工资数据（9,958条记录）

### 代码文件
- `salary_analysis.py` - 主分析脚本，执行所有分析任务

### 输出文件
- `分析报告.md` - 完整的分析报告（Markdown格式）
- `analysis_output.txt` - 控制台输出的详细分析结果

### 图表文件
1. `department_avg_salary.png` - 各部门平均工资柱状图
2. `abs_division_avg_salary.png` - ABS部门各division平均工资柱状图
3. `salary_composition.png` - 工资组成占比饼图
4. `grade_salary_relationship.png` - 级别与工资关系图（包含平均工资趋势和人数分布）

## 如何运行

### 环境要求
```bash
pip install pandas matplotlib numpy
```

### 运行分析
```bash
python salary_analysis.py
```

运行后将：
1. 在控制台输出详细分析结果
2. 生成4张PNG格式的图表
3. 显示图表窗口（关闭后程序结束）

## 分析内容

### 任务1：工资统计
- ✅ 统计最低10位雇员工资
- ✅ 统计最高10位雇员工资
- ✅ 计算最高与最低工资差额

### 任务2：部门分析
- ✅ 计算各department的平均工资
- ✅ 用直方图展示部门平均工资
- ✅ 计算ABS部门各division的平均工资
- ✅ 用直方图展示ABS部门division平均工资

### 任务3：工资组成和级别分析
- ✅ 计算工龄工资在总工资中的占比
- ✅ 计算加班费在总工资中的占比
- ✅ 分析级别（grade）和总工资之间的关系
- ✅ 计算相关系数并给出结论

## 主要发现

### 工资范围
- 最低工资：$11,147.24
- 最高工资：$280,000.00
- 工资差额：$268,852.76

### 工资组成
- 基本工资：92.81%
- 加班费：6.11%
- 工龄工资：1.09%

### 级别关系
- 相关系数：0.5624
- 结论：级别与总工资呈中等正相关关系

### 部门差异
- 最高平均工资部门：Office of Intergovernmental Relations ($148,087.57)
- 技术和管理部门工资明显高于服务部门

## 数据统计
- 总雇员数：9,958人
- 总部门数：40个
- 平均工资：$84,877.20
- 工资中位数：$81,276.58
- 工资标准差：$33,241.76
